# Discover Iran

A simple tourism website built with HTML and CSS.

## About the Project

**Discover Iran** is my first web development project. It was created to practice HTML and CSS while building a website that introduces selected cultural, historical, and tourist attractions of Iran.

The project focuses on presenting information in a clear, organized, and user-friendly way.

## Motivation

I created this project because I wanted to improve my web development skills and combine my interest in technology with my interest in Iranian culture and history.

## Features

* Introduction to selected tourist attractions in Iran
* Image gallery
* Clear navigation between sections
* Responsive design for different screen sizes
* External links for additional information

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Project Structure

```text
discover-iran/

```

## Challenges and Learning Experience

While developing this website, I learned how to:

* Structure a complete website using HTML
* Design a consistent layout with CSS
* Organize images and website content
* Create navigation links between different sections
* Publish a website using GitHub Pages and Netlify

This project helped me better understand the process of building, testing, and publishing a website. It represents my first step into web development and reflects my interest in digital technologies and information management.

## Future Improvements

In the future, I would like to:
